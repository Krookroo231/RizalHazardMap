Hangman Game

Incorrect Guesses:

MECHANICS:

* Start the game by selecting a random word from a predefined list of words.

* Display the word as underscores.

* If letter guessed is in word, reveal all occurrences

* If letter guessed not in word, increment incorrect count and show hangman drawing.

* Keep track and show incorrect guesses to player.

* Display Congratulations pop-up if player wins by guessing all letters.

* Display Game-over pop-up if player lose

* You can restart the game by clicking new game or restart
